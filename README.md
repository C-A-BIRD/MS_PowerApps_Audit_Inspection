<h2>Path to install app to your environment</h3>
<p>Download the .msapp from the solution folder above.<br />
<br />
Within Power Apps Studio, use the .msapp file using <strong>File &gt; Open &gt;</strong> 
Browse and select the .msapp file you just downloaded.<br />
<br />
Select the Data icon.<br />
<br />
To the left of the screen a horizontal display of icons for example an icon with 
layers representing screens. Beneath is a <strong>‘Plus’</strong> icon below is 
a ‘Data’ icon click this. Select data connections a pick “remove.”<br />
<br />
Select <strong>"Add Data"</strong> and search for SharePoint.<br />
Navigate to the site url that you created for the SharePoint <strong>‘Documents 
&amp; Lists’</strong> in an add back in the Desks and Desks Reservations lists.</p>
